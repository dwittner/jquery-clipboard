Copy & Paste Matrix

  1. Click the Copy button to copy the content of the matrix.
  2. Afterwards you can click into a field of the matrix and paste the content back to the matrix via CTRL+V. But don't be surprised. The values are multiplied by 10.
  3. Use the Paste button to paste the content of the clipboard into the textarea.

If you are using a browser other than internet explorer, the fallback textarea will be displayed to copy the text via CTRL+C. Download the example and sign the applet, if you want to copy without the fallback textarea.

The Paste button only works for browsers other than internet explorer, if you download the example and sign the applet!
